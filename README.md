# NexNova

NexNova is a modern web browser written from scratch in C.

## Goals

- HTML parser
- CSS parser
- DOM
- Layout engine
- Rendering engine
- Networking
- JavaScript support (future)

## Project Status

Under active development.

## Vision

NexNova is part of the Nex ecosystem, alongside NexOS and future Nex projects. The goal is to build a lightweight, educational, and modern browser while learning how browser engines work internally.

Contributions are welcome.
